### INTRODUÇÃO

#### Propósito
O propósito deste aplicativo **AviFácil** é ser um sistema de gestão avícola desenvolvido para dispositivos Android. O objetivo principal é fornecer ao produtor rural uma ferramenta robusta para o acompanhamento técnico de lotes de frangos de corte, garantindo a integridade dos dados mesmo em ambientes com conectividade limitada.

#### Escopo
O sistema abrange a autenticação de usuários, o gerenciamento de perfis de avicultores e suas propriedades, a administração de lotes (abertura, acompanhamento e encerramento) e o registro diário de indicadores de desempenho (mortalidade, consumo de ração e peso). Inclui também uma arquitetura de sincronização automática com a nuvem (Firebase).
